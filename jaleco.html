Jaleco App

Gerenciamento de Agenda e Prontuário para Médicos e Pacientes.

A plataforma da Jaleco permite que os médicos acessem sua agenda de consultas e prontuário em seu laptop, tablet ou smartphone. Os pacientes desfrutam da conveniência de agendar uma consulta e acompanhar seu tratamento no conforto de sua própria casa, enquanto ainda têm acesso à equipe de atendimento quando precisam. É o melhor dos dois mundos.




Sobre o Projeto

No projeto JalecoApp foi desenvolvido uma API para se comunicar com 2 sistemas de cliente. Para atender a nessecidades da clinica (agendamento de consulta, Agenda do Doutor, Relatorios), temos um cliente WEB, por outro lado temos um cliente mobile, onde os Doutores tem a possibilidade de controlar sua agenda e verificar informações de seus pacientes



Ferramentas

Para construir API do jalecoapp utilizamos algumas ferramentas, dentre elas:

  • Spring boot
  • JPA
  • API Gateway zuul netflix
  • Hibernate
  • JAVA 8
  • Spring cloud
  • Spring Security
  • Postgres
  • SQL
  • Flyway
  • GIT
