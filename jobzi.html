Jobzi

Jobzi é uma plataforma web que reúne candidatos e empresas, a fim de facilitar o recrutamento. Fui responsável em refatorar todo o backend a API do produto e na migração para uma estrutura de micro services.

Quandro ingressei na Jobzi o primero projeto que participei foi a reestruturação do BE (backend), a arquitetura era formada por 2 grandes serviços, 1 serviço em java e outro em NodeJs. Depois de definido a arquitetura de microservices (spring cloud, zuul netflix), iniciamos a migração transformando os 2 grandes serviços em 11 microservices, cada um responsável por parte da solução.




Experiência com Java

Nossos serviços eram todos hospedados na Amazon AWS, tive a oportunidade de aprender várias ferramentas como:

  • Spring boot
  • JPA
  • AWS Lambda (Node, JAVA 8 - EC2
  • API Gateway / Gateway websocket
  • Load Balancer
  • SES (Aws Email)
